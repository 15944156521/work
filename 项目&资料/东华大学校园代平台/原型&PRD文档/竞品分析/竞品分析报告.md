## 一、竞品分析背景

当前东华学生的代业务、二手交易需求，主要通过 “代课微信群、校园 QQ 频道、闲鱼” 等工具满足，但存在信息杂乱、信任缺失、无校园专属场景等痛点。本次分析聚焦这 3 类核心竞品，明确东华 “代” 平台的差异化设计方向。

## 二、竞品选择及基本信息

| 竞品名称       | 产品形态                                      | 核心服务场景                 | 校园用户覆盖情况                        |
| -------------- | --------------------------------------------- | ---------------------------- | --------------------------------------- |
| 学校代课微信群 | 微信社群（4 个 500 人群）                     | 代业务信息发布、二手闲置接龙 | 覆盖约 2000 名学生（核心是大二 / 大三） |
| 校园 QQ 频道   | QQ 垂直社群（分 “代取 / 二手 / 新生” 子频道） | 分场景信息发布、简易互动     | 覆盖约 1500 名学生（含新生群体）        |
| 闲鱼（校园版） | 综合 C2C 平台                                 | 二手交易、跑腿服务           | 覆盖全校园学生，但校园专属功能弱        |

## 三、核心维度竞品对比

### 3.1 核心功能与用户体验对比

| 对比维度     | 学校代课微信群                                               | 校园 QQ 频道                                                 | 闲鱼（校园版）                                            |
| ------------ | ------------------------------------------------------------ | ------------------------------------------------------------ | --------------------------------------------------------- |
| 代业务场景   | 仅文字 / 图片发需求，信息刷屏；无流程约束，代上课合规性无管控 | 分 “代取 / 代签” 子频道，信息分类更清晰；但无需求模板，沟通低效 | 支持 “跑腿” 服务，但无代上课合规场景；价格不透明          |
| 二手交易场景 | 仅文字描述，无标签 / 筛选；无交易担保，纠纷靠 “熟人信任”     | 可附图片，无 “专业 / 年级” 标签；自提信息模糊                | 支持商品标签，但无 “校园专业 / 年级” 专属筛选；有交易担保 |
| 新生服务场景 | 无专属板块，新生需求易被刷屏淹没                             | 有 “新生子频道”，但流程指引零散（仅靠群友分享）              | 无校园新生专属功能                                        |
| 信任保障     | 无（全靠用户自主判断，纠纷无解决渠道）                       | 无（仅靠频道管理员简单审核）                                 | 有交易担保、卖家信誉分，但无校园场景专属信誉体系          |
| 操作效率     | 低（需翻聊天记录找信息）                                     | 中（分频道但仍需刷消息）                                     | 中（流程完善但校园场景匹配度低）                          |

### 3.2 核心劣势（用户痛点）

1. **学校代课微信群**：
   - 信息过载：代取需求 1 小时刷屏超 50 条，用户需翻记录找有效信息；
   - 信任缺失：代上课 / 代取纠纷无解决渠道（调研显示 45% 男生遇过 “接单者失约”）；
   - 无合规约束：代上课场景混杂 “核心课代上” 违规需求，风险高。
2. **校园 QQ 频道**：
   - 无流程闭环：仅做 “信息发布”，无需求确认、服务验收环节；
   - 筛选低效：二手商品无 “专业 / 年级” 标签，新生买教材需反复问版本（调研中 65% 用户需标签筛选）。
3. **闲鱼（校园版）**：
   - 校园属性弱：无代上课合规场景、无新生导航 / 流程指引；
   - 用户不精准：卖家含校外人员，校园自提效率低（调研中 30% 用户吐槽 “自提距离远”）。

## 四、东华 “代” 平台差异化机会总结

基于竞品劣势，结合调研需求，核心差异化方向：

1. **校园专属场景覆盖**：
   - 独家做 “代上课合规场景”（非核心课代签 / 代记笔记），填补微信群 / 闲鱼的合规空白；
   - 新增 “新生精准导航 + 流程指引”，解决 QQ 频道信息零散的痛点。
2. **流程闭环 + 效率提升**：
   - 代业务做 “标准化需求模板”（校区 / 教室 / 时间必填），解决微信群 / QQ 频道的沟通低效；
   - 二手交易强制 “专业 + 年级标签”，满足 65% 用户的精准筛选需求。
3. **信任 + 合规双保障**：
   - 代业务加 “服务凭证上传”（签到照 / 笔记照），解决微信群无信任的问题；
   - 二手交易做 “校园自提点 + 交易担保”，结合闲鱼的担保优势 + 校园本地化场景。
