# 财务类APP原型图标注文档
## 文档说明
| 项       | 内容                                                         |
| -------- | ------------------------------------------------------------ |
| 文档版本 | V1.0                                                         |
| 更新时间 | 2025-12-10                                                   |

## 一、财务仪表盘页
页面定位：APP首页，展示用户核心财务数据及快速操作入口

### 1.1 顶部导航栏模块
| 元素名称 | 元素类型 | 内容/样式                    | 交互逻辑               |
| -------- | -------- | ---------------------------- | ---------------------- |
| 页面标题 | 文本     | “财务仪表盘”（16号字，加粗） | 仅展示，无交互         |
| 通知图标 | 图标     | 铃铛形状                     | 点击后弹出未读通知列表 |
| 设置图标 | 图标     | 齿轮形状                     | 点击后跳转至「设置页」 |


### 1.2 用户问候&当前金额概览区
| 元素名称     | 元素类型 | 内容/样式                                               | 交互逻辑                                 |
| ------------ | -------- | ------------------------------------------------------- | ---------------------------------------- |
| 用户问候语   | 文本     | “早上好!用户”（14号字）                                 | 依据时段自动切换（早/中/晚）             |
| 概览标题     | 文本     | “这是您的财务概括”（12号字）                            | 仅展示                                   |
| 当前金额展示 | 数据文本 | “¥2,300,00.25”（24号字，蓝色）                          | 实时同步用户账户总金额                   |
| 刷新图标     | 图标     | 环形箭头形状                                            | 点击后重新拉取最新账户数据               |
| 收支小节卡片 | 信息卡片 | 分“本月收入/本月支出/节余”，展示对应金额（示例：¥0.00） | 点击卡片跳转至「消费分析页」对应维度详情 |


### 1.3 核心操作按钮区
| 元素名称   | 元素类型 | 内容/样式               | 交互逻辑                     |
| ---------- | -------- | ----------------------- | ---------------------------- |
| “添加”按钮 | 圆形按钮 | 加号图标+“添加”文本     | 点击后跳转至「添加账单页」   |
| “收入”按钮 | 圆形按钮 | 向上箭头图标+“收入”文本 | 点击后进入收入类账单添加流程 |
| “支出”按钮 | 圆形按钮 | 向下箭头图标+“支出”文本 | 点击后进入支出类账单添加流程 |


### 1.4 月度收支趋势区
| 元素名称     | 元素类型 | 内容/样式                      | 交互逻辑                     |
| ------------ | -------- | ------------------------------ | ---------------------------- |
| 维度切换标签 | 选项卡   | “年/月/日”三个标签             | 点击标签切换图表的时间维度   |
| 收支柱状图   | 图表     | 展示对应时间维度的每日收支数据 | 点击柱形弹出该日收支明细弹窗 |


### 1.5 最近交易区
| 元素名称 | 元素类型 | 内容/样式                                               | 交互逻辑                           |
| -------- | -------- | ------------------------------------------------------- | ---------------------------------- |
| 区域标题 | 文本     | “最近交易”+“查看全部”                                   | 点击“查看全部”跳转至交易记录列表页 |
| 交易条目 | 列表项   | 包含分类图标（如餐饮）、时间、金额（支出为负/收入为正） | 点击条目进入账单编辑页             |


### 1.6 底部功能导航栏
| 元素名称                          | 元素类型 | 内容/样式        | 交互逻辑                   |
| --------------------------------- | -------- | ---------------- | -------------------------- |
| 功能图标（仪表盘/分析/添加/我的） | 图标导航 | 四个功能入口图标 | 点击对应图标跳转至对应页面 |


## 二、消费分析页
页面定位：展示收支数据的多维度分析结果，辅助用户了解消费习惯

### 2.1 顶部导航栏模块
| 元素名称 | 元素类型 | 内容/样式                  | 交互逻辑                                 |
| -------- | -------- | -------------------------- | ---------------------------------------- |
| 返回图标 | 图标     | 向左箭头                   | 点击后返回上一级页面（默认返回仪表盘页） |
| 页面标题 | 文本     | “消费分析”（16号字，加粗） | 仅展示                                   |


### 2.2 时间筛选区
| 元素名称   | 元素类型 | 内容/样式                   | 交互逻辑                                                   |
| ---------- | -------- | --------------------------- | ---------------------------------------------------------- |
| 时间选项卡 | 选项卡   | “本月/上月/全年”+“筛选”按钮 | 点击选项卡切换数据时间维度；点击“筛选”弹出自定义时间选择器 |


### 2.3 收支概览卡片区
| 元素名称 | 元素类型 | 内容/样式                                                   | 交互逻辑                 |
| -------- | -------- | ----------------------------------------------------------- | ------------------------ |
| 收入卡片 | 数据卡片 | 展示对应时间维度的总收入+同比上月变化（示例：“↑50%较上月”） | 点击卡片查看收入分类明细 |
| 支出卡片 | 数据卡片 | 展示对应时间维度的总支出+同比上月变化（示例：“↓30%较上月”） | 点击卡片查看支出分类明细 |


### 2.4 支出分类饼图区
| 元素名称 | 元素类型  | 内容/样式                                      | 交互逻辑                             |
| -------- | --------- | ---------------------------------------------- | ------------------------------------ |
| 饼图标题 | 文本      | “支出分类”                                     | 仅展示                               |
| 分类饼图 | 图表      | 展示各支出分类占比（示例：餐饮35%、娱乐25%等） | 点击饼图分区弹出该分类的支出明细列表 |
| 分类图例 | 文本+色块 | 对应饼图的分类名称+占比                        | 点击图例同步选中饼图对应分区         |


### 2.5 月度趋势图表区
| 元素名称   | 元素类型 | 内容/样式                      | 交互逻辑                        |
| ---------- | -------- | ------------------------------ | ------------------------------- |
| 趋势标题   | 文本     | “月度趋势”+“收入/支出”切换按钮 | 点击“收入/支出”切换图表数据类型 |
| 趋势折线图 | 图表     | 展示对应时间维度的收支趋势     | 点击折线节点弹出该日收支详情    |


### 2.6 AI洞察提示区
| 元素名称 | 元素类型 | 内容/样式                                                    | 交互逻辑                 |
| -------- | -------- | ------------------------------------------------------------ | ------------------------ |
| 提示卡片 | 信息卡片 | 展示AI生成的消费建议（示例：“本月餐饮支出较上月增加了10%...） | 点击卡片查看完整洞察报告 |


## 三、添加账单页
页面定位：用户录入收入/支出账单的操作页面

### 3.1 顶部导航栏模块
| 元素名称 | 元素类型 | 内容/样式        | 交互逻辑                                                     |
| -------- | -------- | ---------------- | ------------------------------------------------------------ |
| 返回图标 | 图标     | 向左箭头         | 点击后返回上一级页面，未保存的内容弹窗提示是否放弃           |
| 页面标题 | 文本     | “添加账单”       | 仅展示                                                       |
| 保存按钮 | 按钮     | “保存”（14号字） | 点击后校验必填项（金额、分类），校验通过则提交账单并返回仪表盘页；不通过则提示缺失项 |


### 3.2 账单金额输入区
| 元素名称   | 元素类型 | 内容/样式        | 交互逻辑                                                 |
| ---------- | -------- | ---------------- | -------------------------------------------------------- |
| 金额输入框 | 输入框   | 默认占位符“0.00” | 仅支持数字+小数点输入，自动格式化金额（如1000→1,000.00） |


### 3.3 分类选择区
| 元素名称   | 元素类型  | 内容/样式                    | 交互逻辑                                 |
| ---------- | --------- | ---------------------------- | ---------------------------------------- |
| 分类标签组 | 图标+文本 | 包含“餐饮、交通、工资”等分类 | 点击分类标签选中（单选），选中后标签高亮 |


### 3.4 备注输入区
| 元素名称   | 元素类型 | 内容/样式                  | 交互逻辑              |
| ---------- | -------- | -------------------------- | --------------------- |
| 备注输入框 | 文本框   | 占位符“请输入备注（选填）” | 支持最多200字文本输入 |


## 四、设置页
页面定位：用户管理账户、系统配置的功能页面

### 4.1 顶部导航栏模块
| 元素名称 | 元素类型 | 内容/样式              | 交互逻辑               |
| -------- | -------- | ---------------------- | ---------------------- |
| 页面标题 | 文本     | “设置”（16号字，加粗） | 仅展示                 |
| 帮助图标 | 图标     | 问号形状               | 点击后跳转至帮助中心页 |


### 4.2 用户信息展示区
| 元素名称     | 元素类型 | 内容/样式                    | 交互逻辑                           |
| ------------ | -------- | ---------------------------- | ---------------------------------- |
| 用户头像     | 头像     | 展示用户上传的头像           | 点击后进入头像上传/更换页面        |
| 用户名       | 文本     | 示例：“张晓明”               | 仅展示                             |
| 用户邮箱     | 文本     | 示例：“zhangxiaoming@qq.com” | 点击“编辑信息”跳转至个人信息编辑页 |
| 编辑信息按钮 | 按钮     | “编辑信息”                   | 点击跳转至个人信息编辑页           |


### 4.3 账户设置区
| 元素名称       | 元素类型 | 内容/样式             | 交互逻辑                    |
| -------------- | -------- | --------------------- | --------------------------- |
| 支付设置选项   | 列表项   | “支付设置”+箭头图标   | 点击后进入支付方式管理页    |
| 银行卡管理选项 | 列表项   | “银行卡管理”+箭头图标 | 点击后进入银行卡绑定/解绑页 |
| 密码设置选项   | 列表项   | “密码设置”+箭头图标   | 点击后进入密码修改页        |


### 4.4 通知设置区
| 元素名称     | 元素类型 | 内容/样式        | 交互逻辑              |
| ------------ | -------- | ---------------- | --------------------- |
| 消息通知开关 | 开关控件 | 默认开启（绿色） | 点击切换开启/关闭状态 |
| 消费提醒开关 | 开关控件 | 默认关闭（灰色） | 点击切换开启/关闭状态 |
| 账单提醒开关 | 开关控件 | 默认关闭（灰色） | 点击切换开启/关闭状态 |


### 4.5 关于我们区
| 元素名称     | 元素类型 | 内容/样式                               | 交互逻辑                               |
| ------------ | -------- | --------------------------------------- | -------------------------------------- |
| 关于我们选项 | 列表项   | “关于我们”+箭头图标                     | 点击后进入APP介绍页                    |
| 隐私政策选项 | 列表项   | “隐私政策”+箭头图标                     | 点击后查看隐私政策文档                 |
| 版本更新选项 | 列表项   | “版本更新”+当前版本（示例：“版本v1.1”） | 点击后检测更新，有新版本则弹出更新提示 |
